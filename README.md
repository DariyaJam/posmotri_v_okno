Лендинг “Посмотри в окно”

Интерактивный интерфейс для просмотра видео о выбранном городе.

Реализовала:
1) вёрстку по макету Figma
2) стилизацию форм и интерактивных элементов
3) состояния элементов через CSS-псевдоклассы
4) вёрстку интерфейса для динамического обновления контента

Стек: HTML, CSS, JavaScript

Проект опубликованный на gh-pages: https://dariyajam.github.io/posmotri_v_okno/
